## PyCord Features
- Emoji Support (no Custom Emoji)
- Multi-Language Support
- Low CPU Usage
- Low RAM Usage (depends on how many servers it's in)
- Reply to Users
- Leave Servers
- Handles an average twitter user's messages (can display long messages without breaking the interface)
